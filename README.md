# JARVIS

JARVIS is a personal AI assistant created by Aditya.

## Features

- Flask web interface
- Groq AI brain
- Voice input
- Optional voice output
- Light and dark mode
- Google/YouTube search
- Weather, news, cricket score, and pincode search routing
- Opens apps like Chrome, ChatGPT, WhatsApp, Codex, Gmail, Spotify, VS Code, Notepad, and Calculator
- Opens folders like Downloads, Documents, and Desktop
- Shutdown confirmation safety
